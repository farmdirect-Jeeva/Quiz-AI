## ✨ Features

- 🤖 **AI-Generated Questions** — Fresh, unique MCQs every time using Groq AI (llama-3.3-70b)
- 📚 **7 Topics Covered** — Polity, Economy, History, Geography, Science, Environment & Tamil Nadu
- 🔢 **Flexible Quiz Length** — Choose 5, 10, 15, or 20 questions per session
- ✅ **Instant Answer Feedback** — Green for correct, red for wrong — shown immediately
- 💡 **Explanations for Every Question** — Learn why the answer is correct after each question
- 📊 **Detailed Score Report** — Score circle, correct/wrong count, percentage breakdown
- 🔍 **Full Review Mode** — Go through every question with your answers after the quiz
- 🌙 **Dark / Light Theme** — Toggle anytime, preference saved automatically
- 📱 **Mobile Friendly** — Works perfectly on all screen sizes
- ⚡ **Zero Dependencies** — Pure HTML, CSS & JavaScript — no frameworks, no npm

## 🚀 How to Use

### Step 1 — Get a Free Groq API Key
1. Visit [console.groq.com](https://console.groq.com)
2. Sign up — free, no credit card needed
3. Go to **API Keys → Create API Key**
4. Copy the key — it starts with `gsk_`

### Step 2 — Open the App
1. Open `quiz.html` in any browser
2. Paste your Groq API key and click **Save**
3. Select your **Topic** and **Number of Questions**
4. Click **Generate Quiz** and start answering!

### Step 3 — Review & Improve
- After the quiz, see your **score and percentage**
- Click **Review Answers** to go through every question
- Retry as many times as you want — AI generates fresh questions every time!
